ESP32-Cam mit OV3660


ESP32-Cam mit OV3660 als Kamera Lok


Funktionsweise:
ein Videostream wird von der Kamera über WLAN übertragen und kann in einem Browser (Firefox, Edge, Chrom,..) oder VLC, ... angezeigt werden.
Zeitgleich kann immer nur eine Verbindung zur ESP32-CAM aufgebaut werden. (Browser oder VLC, ...)

Inbetriebnahme: sehr einfach, mit der aktuellen Arduino-IDE, unter Beispiele – ESP – Camera – CameraWebServer

Die Konfigurationen des Moduls kann über den Browser oder Script (siehe ausführliche Doku) erfolgen.

Zur Stromversorgung der ESP32-CAM verwende ich ein Powerbank, welche in einem Waggon liegt, das Kamera Modul wird von eine Lok geschoben.

das kleine ESP32-CAM Modul gibt es günstig auf den bekannten Web Shops.
