Raspberry Pi - USV

Eine unterbrechungsfreie Stromversorgung USV für einen Raspberry Pi3 oder Pi4

Raspberry Pi USV selber bauen, für die primäre Strom Versorgung nutze ich ein altes Notebook Netzteil (Ausgangsspannung 12V - 20V),
für den USV - Akkubetrieb, nutze ich den Akku (7,2V) eines alten Notebooks, es könnte auch ein kleiner 12V Blei Akku sein.
Ein "Step Down Regler Modul" erzeugt Ausgangsseitig eine Spannung von 5.1V, welche nun über die GPOI Steckerleiste den Pi versorgt.


Bei Netzspannungsausfall wird die Statusänderung über einen Optokoppler an den Raspberry Pi gemeldet (GPIO18 = wiringpi Pin1),
ein Shell Script Pi_USV.sh löst nach 2 Minuten, falls die Netzspannung nicht wieder kommt einen shutdown das Pi aus.
Mit den Schalter kann der Pi aus- und wieder eingeschalten werden.

Voraussetzung für die Funktion des Script Pi_USV.sh ist die Installation von WiringPi
Siehe: WiringPi installieren & Pinbelegung (Raspberry Pi) wiringpi https://tutorials-raspberrypi.de/wiringpi-installieren-pinbelegung (externer Link)
oder: https://github.com/wiringpi/wiringpi (externer Link)
