DCC Sniffer

DCC Sniffer zum Mitlesen der DCC Befehle "RailData" auf dem s88-N Bus

für den DCC Sniffer verwende ich einen Arduino Nano, welcher die Daten über USB zum Mitlesen mittels PuTTY auf dem PC bereitstellt, mit 115200 baud gab es sporadische Übertragungsfehler, deshalb ich eine Übertragungsgeschwindigkeit von 57600 baud verwende.

Quelle: Philipp Gahtow - DCC Sniffer (externer Link)

- DCC_Sniffer.ino,
- DCC_Sniffer2.ino, von mir bevorzugt zur schnelle Überprüfung.
zusätzlich wird die Libary "TimerOne" benötigt, die aktuelle Libary funktioniert bei mir nicht, hier ein ältere Version: Libary_fuer_DCC_Sniffer-TimerOne.zip mit dieser älteren "TimerOne" Libary funktionieren das Mitlesen auf einen Arduino Nano.
