DCC Sniffer

DCC Sniffer zum Mitlesen der DCC Befehle "RailData" auf dem s88-N Bus

für den DCC Sniffer verwende ich einen Arduino Nano, welcher die Daten über USB zum Mitlesen (57600 baud) mittels PuTTY auf dem PC bereitstellt.

Quelle: Philipp Gahtow - DCC Sniffer (externer Link)

- DCC_Sniffer.ino,
- DCC_Sniffer2.ino, von mir bevorzugt zur schnelle Überprüfung.
zusätzlich wird die Libary "TimerOne" benötigt, die aktuelle Lib funktioniert bei mir nicht, hier ein ältere Version: Libary_fuer_DCC_Sniffer-TimerOne.zip
